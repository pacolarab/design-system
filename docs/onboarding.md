# Onboarding — Design System Hanzo Studio
## Lo que necesitas
- Una cuenta de Claude en claude.ai
- Acceso al repositorio de GitHub (pídele acceso a Paco)
---
## Paso 1 — Crear cuenta en Claude
Ve a claude.ai y crea una cuenta gratuita o de pago.
---
## Paso 2 — Crear el Project en Claude
1. En el panel lateral haz clic en Projects → New Project
2. Nómbralo "Design System Hanzo"
3. En Instrucciones pega el contenido del archivo docs/workflow.md de este repositorio
---
## Paso 3 — Conectar el repositorio
1. En Archivos haz clic en +
2. Selecciona GitHub
3. Conecta el repositorio pacolarab/design-system
---
## Paso 4 — Ver los componentes
Sin instalar nada puedes ver la librería en:
https://69a55ed0c390888b2b3b09b2-wyypduitwt.chromatic.com/
---
## Paso 5 — Trabajar con el código (opcional)
1. Clona el repositorio: git clone https://github.com/pacolarab/design-system
2. Instala dependencias: npm install
3. Arranca Storybook: npm run storybook
